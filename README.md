# Mobile App Development - Assignments & Projects

## 📌 About

This repository contains all assignments, activities, and projects completed during the **Mobile App Development** course using **React Native**. It includes two complete apps, three assignments, a mid-term project, and a quiz module.

---

## 📁 Repository Structure

| Type | Name | Description |
|------|------|-------------|
| **Activity** | Movie App | Browse and view movie details |
| **Activity** | Bakery App | Online bakery ordering interface |
| **Assignment** | AsyncStorageApp | Local data persistence with AsyncStorage |
| **Assignment** | JuicyJoyRide | Juice delivery app concept |
| **Assignment** | FirebaseApp | Firebase integration (Auth/Firestore) |
| **Mid-Term** | StyleHub | Fashion shopping app (like Daraz) |
| **Quiz** | LoginModule | User authentication module |
